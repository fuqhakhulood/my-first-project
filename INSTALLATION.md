# تعليمات التثبيت والإعداد

## متطلبات النظام
- تأكد من تثبيت [Node.js](https://nodejs.org/en/) (الإصدار 14 أو أعلى)
- تأكد من تثبيت [npm](https://www.npmjs.com/) (المدير الحِزمي لـ Node.js)

## خطوات التثبيت
1. قم باستنساخ المستودع:
   ```bash
   git clone https://github.com/fuqhakhulood/my-first-project.git
   ```

2. انتقل إلى دليل المشروع:
   ```bash
   cd my-first-project
   ```

3. قم بتثبيت الاعتماديات:
   ```bash
   npm install
   ```

## تشغيل المشروع

للبدء في استخدام المشروع، قم بتشغيل:
```bash
npm start
```

## الإعدادات الإضافية

يرجى مراجعة ملف `config.js` لتخصيص إعدادات المشروع الإضافية حسب الحاجة.

## الدعم

إذا كان لديك أي استفسارات أو واجهت مشاكل، لا تتردد في فتح قضية في قسم المشاكل (Issues) لهذا المستودع.
